对比Java

Kotlin解决了一些Java中难搞的问题

Kotlin 修复了Java中一系列长期困扰我们的问题

  • 类型系统控制了空引用的发生.
  • 没有原始类型
  • Kotlin中数组是不变的
  • 相对于Java的SAM-conversions，Kotlin有更加合适的函数类型
  • 使用没有通配符的site variance
  • Kotlin 没有检查异常

Java有但是Kotlin没有的东东

  • 检查异常
  • 原始类型 ，不是类
  • 静态成员
  • 未私有化字段
  • 通配符类型

Kotlin有但是Java没有的东东

  • Lambda 表达式 + 内联函数 = 高性能自定义结构
  • 扩展函数
  • Null 安全性
  • 智能类型转换
  • 字符串模板
  • 属性
  • 主构造方法
  • 一级委托
  • 变量和属性类型的类型推断
  • 单例
  • 声明点变化 & 类型预测
  • 范围表达式
  • 运算符重载
  • 同伴对象
  • Data classes
  • Separate interfaces for read-only and mutable collections
