Tutorials

A series of complete step-by-step tutorials to walk you through how to accomplish different tasks in Kotlin, from Getting Started to writing Android applications and more. If you have a tutorial you’d like featured here, please let us know. We’ll be happy to host it and give credit!

  • If you like a “hands-on” learning, try Kotlin Koans online to get familiar with Kotlin.
  • If you are using the command line compiler, start with Working with the Command Line Compiler and then work your way through the Kotlin Koans.
  • If you are using IntelliJ IDEA, start with Getting Started and then work your way through the Kotlin Koans.
